Book Database with Django Rest Framework

Populating data from bookshelf images using advanced image recognition

GitHub Repo

Overview

The Book Database project leverages Computer Vision to create a comprehensive database of books. Rather than manual data entry, this system uses OpenCV and Google Cloud endpoints to extract book titles and other relevant information directly from photographs of bookshelves.

How It Works

  • Users upload images of their bookshelves.
  • Image recognition software scans and identifies book spines, extracting titles and authors.
  • The extracted data is then looked up in Google Books.
  • Finally the comprehensive response from Books API is stored in the database.
  • Used Bookstore X can now search, filter, and manage their book catalogue online with ease.

Potential Applications

This innovative approach to data entry has numerous potential applications:

  • Personal Libraries: Individuals can quickly catalog their personal book collections.
  • Bookstores: Store owners can maintain real-time inventory and organize store layouts.
  • Libraries: Libraries can expedite their inventory processes, especially during returns and new acquisitions.
  • Research: Academics can catalog research materials efficiently for easy reference.
