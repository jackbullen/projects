Course Database

Fall 2023 and Spring 2024 UVic course data API deployed to GCP App Engine.

Overview

This project is about creating a comprehensive database for UVic courses for the Fall 2023 and Spring 2024 semesters. The data is hosted via an API deployed to Google Cloud Platform's App Engine.

Key Features

  • Real-time course data
  • Powerful search capabilities
  • Integration with GCP App Engine

Technologies Used
