Course Database

Fall 2023 and Spring 2024 UVic course data API deployed to GCP App Engine.

GitHub Repo

Overview

This project is about creating an API endpoint for University of Victoria course data including schedules for class sessions. The data is hosted in Cloud SQL and Django is in App Engine.

Key Features

  • Detailed query parameters
  • Latitude and longitude of buildings
  • Sections, Courses, Instructors, and their Relations

Technologies Used

GCP: Cloud SQL/Cloud Proxy/App Engine

Python: Django with Rest Framework
